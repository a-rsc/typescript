## Terminal

Ejecutar los siguientes comandos por el terminal:

`cd chapter?` (sea el capítulo que sea)
`tsc --watch`
`node dist/?.js`

---
## Comentarios:

Cuando se empieza un capítulo `tsc --init` y se copia el contenido de tsconfig.json de cualquier otro capítulo.
